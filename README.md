# Pokemon Fire Ash Port for PortMaster (R36S/ArkOS)

## Instructions
1. Download Pokemon Fire Ash from: https://reliccastle.com/resources/508/
2. Copy all game files to PokemonFireAsh-game/ folder
3. Copy PokemonFireAsh.sh to /roms/ports/ on your R36S
4. Copy PokemonFireAsh-game/ to /roms/ports/ on your R36S

## Notes
- This port uses mkxp-z engine from Pokemon Reborn port
- Requires PortMaster to be installed on your device
- Based on work by JanTrueno and the mkxp-z project
